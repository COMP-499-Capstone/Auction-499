# Auction Platform
A full stack realtime auction platform built with React, Supabase, Node.js, and Express. The application allows users to browse auction listings, participate in live-streamed auctions, and securely complete payments using Stripe

the platform was built by our software engineering team for our capstone to demonstate our capalbilties in completing a live fullstack website. after the capstone the site was taken down because we saw no future in people using it along with the price of keeping the site up, the database and livekit we better off see this as a demonstation of our skills

# Features
- User authentication and account
- Create and browse auction listings
- Timed listings and livestreames
- Real-time bidding functionality
- Live-streamed auctions powered by LiveKit
- Secure payment processing with Stripe
- Backend service for payment handling and secure server logic
- Role-based access for Supabase tables

# Tech Stack
Frontend
- React
- Javascript
- CSS
Backend
- Node.js
- Express.js
Database & Auth
- Supabase(PostgreSQL + Auth)
Third-Party Integrations
- Stripe(payment processing)
- LiveKit(real-time livestreaming)

# Architecture Overview 
- React handles UI, routing, and client-side state
- Supabase manages authentication and relational database storage
- Node/Express backend handles:
    - Stripe payment intent creation
    - Secure server-side operations
    - Enviornment-protected API keys
- LiveKit enables real-time video streaming during auctions
